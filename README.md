## Description

A fully functional RESTful API built with **NestJS**, using **TypeORM** to manage data with **PostgreSQL**. This project follows enterprise-grade best practices such as layered architecture, DTOs, environment configuration, validation, and more.

## ðŸš€ Features

- Full CRUD operations for managing employees
- Data validation using `class-validator`
- Clean architecture using Controllers, Services, Repositories, and Mappers
- Environment-based configuration support
- Enum support for fields like department and designation
- Uses PostgreSQL via Docker container (no need to install locally)
- Built with TypeScript and supports DTOs for strong typing
- Exception handling for meaningful error responses
- Ready for future extensions (e.g., auth, GraphQL, etc.)

## Project setup


## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash

$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
